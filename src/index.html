UNFCCC CRF National Emission Inventory Sunbursts - Belgium 2018

Below graph displays the Belgian UNFCCC CRF Emission Inventory Data for the year 2018 (submission 2020), using hierarchical data from the European Environmental Agency (EEA, May 2020)

It displays the emissions in Gg CO2eq for all greenhouse gasses combined, for all non-empty categories except LULUCF and international bunkers. 

The application in development can be found on GitHub and CodeSandbox. Feel free to contribute / do a pull request ! The purpose is to create an interactive UNFCCC CRF Sunburst-explorer for all countries that reported their emission inventory to the UNFCCC, using a country- and year-selection functionality and displaying historical emission line-graphs for each main inventory category. More feature ideas are listed in the GitHub Issues.
